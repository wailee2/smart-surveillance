# Demo Videos

Place the following two video files in this folder before sharing the repository:

| File | Description |
|------|-------------|
| `sample_input.mp4` | A short (5–30 second) raw traffic video used as the app input |
| `sample_output.mp4` | The annotated output produced by the app from the input above |

These files are excluded from version control by `.gitignore` because they can be large.
Host them on Google Drive / YouTube / GitHub Releases and link to them in the main README.
